# nom: popover précisant les consommations

# objectif

Au clic sur l'élément de menu, ouvrir une popover avec à l'intérieur 1 card par account (quand multi-account pour un vendor donné, afficher le compte actif de manière proéminente)

Dans chaque card, on affiche les usages de chaque account:
- pour les time-window: une progressbar avec libellé + pourcentage consommé + durée d'ici au prochain reset + date du prochain reset + indicateur de consommation théorique (en fonction de la taille de la fenetre + du temps restant d'ici au prochain reset)
- pour les pay-as-you-go: le montant actuellement consommé

Utiliser le skill impeccable pour faire une jolie UI avec une bonne UX



Planifier les autres features suivantes (à détailler ensemble):
- Afficher des erreurs de status monitor pour tous les vendors dont on a au moins un account (pour claude: https://status.claude.com/)
- Rajouter de la couleur sur les métriques (dans toolbar (pastille) + progress bars) : en fonction du ratio consommation réelle / consommation théorique : <0.7 c'est vert, [0.7, 0.9[ c'est bleu, [0.9, 1[ c'est jaune, [1, 1.2[ c'est orange, [1.2, 1.6[ c'est rouge, >1.6 c'est noir
- Ajouter un cog dans la popover, permettant d'ouvrir une fenetre de configuration de l'app
- Ajouter un bouton "buy me a coffee"
